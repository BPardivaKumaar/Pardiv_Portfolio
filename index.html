Intelligent Door Lock System

This project presents the development of an IoT-enabled smart door security system using an ESP32 microcontroller, designed as part of my 4th-year first-semester work (2025). It includes a servo smart lock, PIR motion detection, a DHT22 sensor, keypad authentication, and an LCD for real-time updates. Using MQTT and a Node-RED dashboard, users can remotely monitor the system and lock or unlock the door. The design supports extensions like an ESP32-CAM for video and voice communication. This open-source, customizable, and cost-efficient solution highlights my skills in embedded systems, IoT communication, and practical automation.

SURVILLANCE ROBOT

This project, which I completed in the 2nd year of my B.Tech, is a Wi-Fi controlled robotic car built using an ESP32-CAM and a smartphone. It is designed for simple surveillance, with a camera that streams live video to a web browser. The same webpage also lets the user control the robot's movement-forward, backward, left, and right. The setup includes an ESP32-CAM, L293D motor driver, two DC motors, and a basic chassis. Commands are sent wirelessly from the phone, and the robot moves accordingly while sharing real-time video. This makes the system useful for basic spying, monitoring, and remote observation tasks.
